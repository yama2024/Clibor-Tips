クリップボード拡張ソフト「Clibor」の使い方をインタラクティブに学ぼう

1. 基本のキ！クリップボード履歴の呼び出しと貼り付け

このセクションでは、Cliborの最も基本的な操作である履歴の呼び出しと貼り付け方法を学びます。この操作をマスターするだけで、日々のコピー＆ペースト作業が劇的に効率化されます。

履歴を呼び出す

キーボードの Ctrl キーを素早く2回押します。すると、マウスカーソルの位置にCliborのメイン画面が表示され、コピー履歴が一覧で表示されます。

Ctrl + Ctrl → 📋 Cliborメイン画面

履歴を選択して貼り付ける

キーボード操作:

1. ↑ ↓ キーで履歴を選択

2. Enter キーで貼り付け

マウス操作:

貼り付けたい履歴をダブルクリックします。

2. よく使う文章は「定型文」に登録しよう

メールの署名や挨拶文など、頻繁に使う文章を「定型文」として登録する方法です。一度登録すれば、いつでも瞬時に呼び出して貼り付けることができます。

定型文を登録する

  1. Cliborメイン画面で「定型文」タブをクリック。
  2. グループ（例: 共通）の上で右クリック。
  3. 「定型文の新規登録」を選択。
  4. 文章を入力して「OK」をクリック。

定型文を貼り付ける

操作はクリップボード履歴と同じです。「定型文」タブで貼り付けたい文章を選び、Enterキーかダブルクリックで貼り付けます。

✨ Tips

定型文はグループ分けが可能です。「ビジネス」「プライベート」のように分類すると、さらに管理しやすくなります。

3. 連続貼り付けの達人になる「FIFO/LIFOモード」

複数の項目を順番に貼り付けたい時に絶大な効果を発揮するのが「FIFO/LIFOモード」です。下のシミュレーションで、どのように動作するのかを体感してみてください。

使い方

  1. Cliborメイン画面を開く。
  2. Shift を押しながら貼り付けたい項目を複数選択。
  3. Shift を押したまま Enter を押す。
  4. 貼り付けたい場所で Ctrl + V を押すたびに、順番に貼り付けられる。

動作シミュレーション

選択した項目リスト:

  • 1. 氏名
  • 2. 住所
  • 3. 電話番号

4. 目的の履歴をすぐ見つける「インクリメンタルサーチ」

クリップボード履歴が増えても、この検索機能を使えば一瞬で目的のテキストを見つけ出せます。Cliborのメイン画面を開いて、そのままキーワードを入力するだけです。

📝

履歴の編集

編集したい履歴の上で右クリックし、「クリップボードの編集」を選択すると内容を修正できます。

🗑️

履歴の削除

削除したい履歴を選択して Delete キーを押すと、個別に削除できます。

⚙️

ホットキーの変更

タスクトレイのCliborアイコンを右クリックし「設定」を開くと、メイン画面を呼び出すキーなどをカスタマイズできます。
